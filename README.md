# LxGuard.AI Backend

This Space hosts the backend API for LxGuard.AI, built with FastAPI and running in a Docker container.

## Configuration

- **SDK**: Docker
- **Port**: 7860
- **Database**: External (Supabase)
- **Cache**: External (Upstash)
